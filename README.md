# MaTouch_AMOLED_AI_CAMERA

## Introduction

The "MaTouch_AMOLED_AI_CAMERA" is a high-definition integrated display, touch and vision sensing product built around the ESP32-S3 chip with built-in Wi-Fi function. It retains a high-quality 1.8-inch AMOLED screen featuring outstanding high contrast ratio and rich color saturation, delivering ultra-clear, vivid and delicate display effects for diverse visual interaction scenarios. Equipped with the professional CTS820 touch chip, it achieves more sensitive, accurate and stable touch response, effectively optimizing daily operation and human-computer interaction experience. Furthermore, the product integrates a high-performance OV3660 CMOS camera module with 3-megapixel high resolution, supporting multiple image output formats and automatic image optimization functions including automatic exposure control and automatic white balance. With excellent low-light imaging performance and high pixel fidelity, it enables clear and stable image capture, visual recognition and AI vision analysis, greatly expanding the product’s application boundaries in embedded vision, intelligent monitoring, interactive shooting and AI scenario development.
